# 🏦 BankingApp - Console Banking System

A clean, feature-rich, and well-structured **C# Console Banking Application** built with best practices in mind. Perfect for demonstrating OOP, separation of concerns, file persistence, and admin functionalities.

![C#](https://img.shields.io/badge/C%23-239120?style=for-the-badge&logo=c-sharp&logoColor=white)
![.NET](https://img.shields.io/badge/.NET-512BD4?style=for-the-badge&logo=dotnet&logoColor=white)

## ✨ Features

### User Features
- ✅ Create new bank accounts with proper input validation
- ✅ Deposit and Withdraw money
- ✅ Transfer funds between accounts
- ✅ Check account balance
- ✅ View detailed transaction history
- ✅ Pakistani-specific validation (Phone & CNIC format)

### Admin Features
- 🔐 Secure Admin Panel with login
- 👁️ View all customer accounts
- 🔒 Freeze / Unfreeze accounts (prevents transactions)
- 📊 Complete transaction logging

### Technical Features
- 💾 Persistent data storage using JSON
- 🛡️ Input validation & error handling
- 📁 Clean layered architecture (Models, Interfaces, Services)
- 🔄 In-memory + File-based persistence
- 🧹 Proper `.gitignore` setup

## 🛠️ Tech Stack

- **Language**: C# 
- **Framework**: .NET (Console Application)
- **Serialization**: System.Text.Json
- **Architecture**: Layered Architecture (Models + Services + Interfaces)
